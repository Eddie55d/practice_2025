- Заместители гендиректора по направлениям:

  - ИТ (Горбунов В. Н.);

**Функциональные подразделения (предположительно, на основе должностей):**

- ИТ‑департамент (инфраструктура, цифровизация, безопасность).

#### 3. Достоинства и недостатки организационной структуры

**Достоинства:**  

1. **Чёткое разделение зон ответственности** — каждый заместитель курирует отдельное направление, что снижает дублирование функций.  
2. **Фокус на цифровизацию** — наличие отдельного заместителя по ИТ и руководителя e‑commerce подчёркивает приоритет цифровых технологий.  
3. **Сильный HR‑блок** — внимание к персоналу как ключевому активу розничной сети.  
4. **Гибкость коммерческой политики** — зам. по коммерции может оперативно реагировать на изменения рынка.  
5. **Географическая масштабируемость** — структура позволяет управлять сетью из 6 000+ аптек через региональные подразделения.

**Недостатки:**  

1. **Высокая централизация** — ключевые решения принимаются на уровне топ‑менеджмента, что может замедлять реакцию на локальные вызовы.  
2. **Риск перегрузки гендиректора** — отсутствие чёткой децентрализации может создавать «узкие места».  
3. **Потенциальные конфликты между блоками** — например, между коммерцией (маржа) и e‑commerce (скидки для онлайн‑продаж).  
4. **Недостаточная прозрачность региональных процессов** — в представленной структуре не видны менеджеры по регионам, что усложняет контроль на местах.  
5. **Зависимость от ИТ‑инфраструктуры** — высокая цифровизация повышает риски при сбоях систем.

#### 4. Уровень развития и внедрения цифровых технологий

**Ключевые достижения:**  

- **Онлайн‑платформа и мобильное приложение** — клиенты могут заказывать лекарства с доставкой или самовывозом.  
- **Интеграция с маркетплейсами** — присутствие на крупных площадках расширяет аудиторию.  
- **Автоматизация складов и аптек** — использование ERP‑систем для учёта товаров и управления запасами.  
- **Электронные рецепты и телемедицина** — частичная цифровизация рецептурного отпуска.  
- **Аналитика данных** — прогнозирование спроса, персонализация предложений, управление остатками.  
- **Кибербезопасность** — защита персональных данных клиентов и платёжной информации.

**Зоны для развития:**  

1. **Полная цифровизация рецептурного оборота** — переход на электронные рецепты во всех регионах присутствия.  
2. **ИИ для прогнозирования спроса** — использование машинного обучения для оптимизации закупок.  
3. **Роботизация складов** — автоматизация логистики для снижения ошибок и сроков доставки.  
4. **Расширение телемедицинских сервисов** — интеграция с клиниками для комплексного обслуживания.  
5. **Улучшение UX/UI онлайн‑платформы** — повышение удобства для пожилых клиентов.

**Вывод:**  
ООО «Ригла» демонстрирует **высокий уровень цифровой зрелости** в ритейле, особенно в e‑commerce и автоматизации. Однако для удержания лидерских позиций требуется дальнейшее внедрение ИИ, роботизированных решений и углублённой аналитики.
